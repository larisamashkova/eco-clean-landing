# Эко-Чистота — Лендинги для продажи экологичных наборов для уборки

## Структура проекта

```
site2/
├── index.html       — Лендинг 1: «Экологичная уборка без вредной химии» (разовая покупка)
├── style.css        — Стили для лендинга 1
├── index2.html      — Лендинг 2: «Чистота без забот — с эко-подпиской» (подписка)
├── index2.css       — Стили для лендинга 2
├── script.js        — Общий JS (анимации, формы, бургер-меню)
```

## Офферы (A/B тестирование)

| Файл | Оффер | Цена |
|------|-------|------|
| `index.html` | Полный набор для уборки | 1 200 ZCN (−15% для первых 50) |
| `index2.html` | Ежемесячная эко-подписка | от 790 ZCN/мес |

## Дизайн-система

- **Primary:** #4CAF50
- **Background:** #FFFFFF / #F5F5F5
- **Text:** #333333
- **Акцент скидки:** #FF6B35
- **Шрифт:** System UI (sans-serif)

## Запуск

Откройте `index.html` или `index2.html` в любом современном браузере.
